# Widget Usage Guide

## How Users Enable/Access the Widget

The widget is **automatically enabled** when you launch Stoic Mirror. Here's how it works:

### Automatic Activation
- ‚úÖ **Widget starts automatically** - The app creates a menu bar (tray) icon when you launch it
- ‚úÖ **Menu bar icon appears** - Look for the Stoic Mirror icon in your macOS menu bar (top right)
- ‚úÖ **No configuration needed** - It's always available

### How to Open the Widget

#### Method 1: Click the Menu Bar Icon (macOS)
- **Left-click** the Stoic Mirror icon in the menu bar
- Widget pops up below the icon
- Click again to toggle (hide/show)

#### Method 2: Right-Click Context Menu
- **Right-click** the menu bar icon
- Select **"Show Widget"** from the menu
- Widget appears below the icon

#### Method 3: Quick Journal Access
- **Right-click** the menu bar icon
- Select **"Quick Journal"** 
- Widget opens directly to journal input

### Widget Features
- Quick journal entry
- View today's schedule summary
- See task completion status
- Minimal, always-accessible interface

### Menu Bar Context Menu Options

When you right-click the menu bar icon, you'll see:
1. **Open Stoic Mirror** - Opens the full app window
2. **Show Widget** - Toggles the widget popover
3. **Quick Journal** - Opens widget directly to journal input
4. **Quit** - Closes the app

### Troubleshooting

**Widget not appearing?**
- Check if the menu bar icon is visible (top right on macOS)
- Try right-clicking the icon and selecting "Show Widget"
- Restart the app if the icon is missing

**Want to hide the widget?**
- Click outside the widget window
- Or click the menu bar icon again to toggle
